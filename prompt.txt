Título:
Criar fundação técnica do monorepo da plataforma B2B de dropshipping, white label e private label
Contexto:
Estamos construindo uma plataforma B2B para uma empresa produtora de bolsas e acessórios. A plataforma permitirá que lojistas vendam produtos via dropshipping, criem marcas próprias em modelo white label e futuramente desenvolvam coleções private label.
A proposta central do produto é permitir que lojistas, influenciadores, boutiques e empreendedores criem uma marca de bolsas e acessórios sem fábrica, sem estoque e sem logística própria.
A arquitetura definida para o projeto é:


Frontend: Next.js com TypeScript.


Backend: Node.js com NestJS e TypeScript.


Banco de dados: Neon PostgreSQL.


ORM: Prisma.


Hospedagem: Render.


Worker: Render Background Worker.


Cron jobs futuros: Render Cron Jobs.


Storage futuro: S3-compatible, preferencialmente Cloudflare R2 ou AWS S3.


Filas futuras: BullMQ com Redis, usando Render Key Value ou Upstash Redis.


Autenticação futura: JWT com refresh token.


Permissões futuras: RBAC.


Deploy: Render via render.yaml.


Objetivo:
Criar a fundação técnica inicial do repositório em formato monorepo, com apps separados para web, api e worker, pacote de database com Prisma, estrutura inicial modular no backend, configuração base do frontend e arquivo render.yaml inicial.
Essa tarefa não deve implementar funcionalidades de negócio completas. O foco é preparar uma base sólida, organizada e extensível para as próximas fases.
